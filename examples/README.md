# Firestarter SDK Examples

Runnable examples showing how to use the SDK in different scenarios.

## Examples

### [basic-usage.ts](./basic-usage.ts)
The simplest possible usage - login, upload, download. **Start here!**

```bash
npm install firestarter-sdk
npx ts-node examples/basic-usage.ts
```

### [react-example.tsx](./react-example.tsx)
Complete React component with file upload/download functionality.

### [wallet-auth.ts](./wallet-auth.ts)
Using wallet addresses to generate deterministic credentials.

## Running Examples

1. Install the SDK:
```bash
npm install firestarter-sdk
```

2. Run any example with ts-node:
```bash
npx ts-node examples/basic-usage.ts
```
